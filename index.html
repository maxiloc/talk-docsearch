DocSearch

The easiest way to add search to your documentation.

Even with the most complete and well-written documentation, finding relevant information is still an issue.

Searching is painful

  • Ctrl + F
  • No typo-tolerance
  • Slow
Not all documentations have a good search, you often have to rely on the good-old Ctrl-F and all its quirks.

Scratching our own itch

  • Hosted search API
  • Fast and relevant
  • Instant search UI
We're Algolia, we build a hosted search API and used to build stuff like this. Surely there is something we could do.

Meet DocSearch

  • Typo-tolerant
  • Ultra-fast
  • Free
So we built DocSearch, a crawler that will read all your HTML documentation and push it to an Algolia index so you can search easily into it.

How does it work?

  1. We configure your settings
  2. We give you a few lines of JavaScript to add to your page
  3. Crawler runs every 24 hours, indexing your HTML pages
